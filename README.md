# Save Coordinates Fabric MC

## Dependencies

Item|Minimum version
-|-
Java|1.8
Minecraft|1.16.5
[Fabric API](https://www.curseforge.com/minecraft/mc-mods/fabric-api/files) | 0.34.2+1.16

## Usage

- Press `H` for menu
- Select `save` to save the coordinate
- Select `list` to view saved coordinates
